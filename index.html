The following visualization analyzes 5,492 satellite images in order to see the relationship between agricultural subsidies, vegetation, GDP and productivity in Malawi, Africa.

It was created with the advisement of the UNDP Regional Bureau for Africa and with the use of Google Earth Engine.

This will eventually be an exploratory tool useful for policymakers which includes all sub-Saharan Africa.
